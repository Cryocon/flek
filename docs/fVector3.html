    fVector3::fVector3 ();
    fVector3::fVector3 (const double z);
    fVector3::fVector3 (const double x, const double y, const double z);

    The constructor for a 3 dimensional vector. For efficiency, fVector3 () is uninitialized. fVector3(const double) takes one initializing argument (for example fVector3 (0) would initialize the vector to <0, 0, 0>. And the third form initializes each element x, y, and z.
